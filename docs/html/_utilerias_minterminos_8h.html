Detailed Description

Declaraciones de funciones auxiliares para el manejo de mintérminos en la simplificación booleana.

Este archivo contiene las funciones necesarias para leer los mintérminos, covertirlos a binario, clasificarlos y combinarlos con el método de Quine-McCluskey e imprimiendo la tabla resultante.

Se usan estructuras de datos que hacen más fácil trabajar paso a paso con mintérminos, agrupándolos según cuántos bits tienen en uno.
