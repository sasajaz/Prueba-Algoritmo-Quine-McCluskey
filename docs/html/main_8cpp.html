int main ( )

Punto de entrada principal del programa.

Lee mintérminos desde entrada, los clasifica, genera las combinaciones de Quine-McCluskey e imprime la tabla de implicantes primos con su cobertura.

Returns
int 0 si la ejecución fue exitosa.

Vector que almacena la clasificación de los mintérminos, en la primera pocisión es el número de bits encendidos que tiene el mintermino el par exterior en su primera posicion tiene otro par con la información del número en bits y en la segunda pocisión tenemos el número del cual surgió el número binario.

Note
Estructura: [iteracionDeLaTabla][minterminosAlmacenados]

IMPORTANTE: A partir de la iteracion 1, los minterminos seran clasificados de forma creciente con referencia a la cantidad de bits encendidos sobre el cual se generó la combinación.

Tabla que representa la cobertura de los mintérminos por los implicantes primos.

1 representa una X, 0 representa un espacio vacio

tablaExpresionesFinales[i][j] = 1 si el implicante i cubre el mintérmino j
